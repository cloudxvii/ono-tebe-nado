**«Оно тебе надо»** - это одностраничный сайт аукциона необычных вещей, в которые когда-то никто не верил. Проект демонстрирует навыки вёрстки сложных макетов с использованием семантической разметки, CSS-гридов, флексбоксов и современных техник стилизации.

## 🚀 Демо

Приложение можно запустить локально, открыв файл `index.html` в браузере.  
*Деплой не предусмотрен, но вы можете склонировать репозиторий и попробовать всё самостоятельно.*

## 🔍 Особенности реализации

- **Грид-сетка** - для шапки использован `display: grid` с тремя колонками (`1fr min-content 1fr`), для секции «Об аукционе» - грид из двух колонок с фиксированной шириной левой колонки.
- **Переиспользуемый оверлей** - один класс `.overlay` применяется и к обложке, и к каждой карточке, что обеспечивает единообразие и упрощает поддержку.
- **Кастомные маркеры списка** - для пунктов меню в шапке задан `list-style: circle`, кроме активного пункта, у которого маркер скрыт.
- **Стилизация ссылок** - у ссылок убрано стандартное подчёркивание, добавлено состояние `:hover` с подчёркиванием; активное состояние (при клике) выделено фоном (для карточек).
- **Фоновые изображения** - каждая карточка имеет уникальный фоновый рисунок, заданный через отдельные классы, изображения масштабируются с `background-size: cover`.
- **Сброс браузерных стилей** - в `global.css` выполнена базовая нормализация: убраны отступы, задана базовая ширина body и семейство шрифтов.
- **Фиксированная ширина** - страница рассчитана на ширину 1100px, что соответствует макету и гарантирует предсказуемое отображение на десктопах.
- **Семантическая вёрстка** - все блоки размечены согласно их смыслу: заголовки (`h1`-`h3`), списки (`ul`), статьи (`article`), контактная информация (`address`).

## 🎯 Цель проекта

Проект был выполнен в рамках учебной программы для отработки навыков:

- вёрстки сложных интерфейсов по макету с использованием гридов и флексбоксов;
- создания переиспользуемых компонентов (оверлей, блок с адресом);
- точного соблюдения дизайна (Pixel Perfect);
- написания семантического HTML и организации CSS-кода.
